Contrastive learning

对比学习是一种学习方法，侧重于通过对比正反两方面的实例来提取有意义的表征。它利用的假设是，在学习到的嵌入空间中，相似的实例应靠得更近，而不相似的实例应离得更远。通过将学习作为一项辨别任务，对比学习允许模型捕捉数据中的相关特征和相似性。

在CLIP, BLIP这些大模型中通常使用对比学习的方式进行预训练。

对比学习主要包括两个分支，分别是监督对比学习（Supervised Contrastive Learning, SCL） 和 **自监督对比学习(Self Supervised Contrastive Learning, SSCL)**。

infoNCE Loss Function

InfoNCE Loss（Noise Contrastive Estimation Loss）是一种用于自监督学习的损失函数，通常用于学习特征表示或者表征学习。不同于交叉熵损失函数，它基于信息论的思想，通过对比正样本和负样本的相似性来学习模型参数。
公式如下：

$
\text{InfoNCE Loss} = - \frac{1}{N}\sum_{i=1}^N \log (\frac{\text{exp}^{(\frac{q_i · k_{i+}}{\tau})}}{\sum_{j=1}^N \text{exp} (\frac{q_i · k_{j-}}{\tau})} )
$

$N$ -> 样本数量
$q_i$ -> 查询样本$i$的编码向量
$k_{i+}$ -> 是与查询样本$i$相对应的正样本的编码向量
$k_{i-}$ -> 是与查询样本$i$不对应的负样本的编码向量
$\tau$ -> 温度系数, 用于调节相似度得分的分布

在训练的过程中，其实就是希望正样本之间的距离越来越近，负样本之间的距离越来越远。

infoNCE loss 在某种情况是是基于交叉熵函数的。

交叉熵损失函数:

$
\text{CrossEntropy(P,Q)} = - \sum_{i} P(i) · log(Q(i))
$
